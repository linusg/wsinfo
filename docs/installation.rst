.. _installation:

Installation
============

The |wsinfo| library is available on PyPI_, so you can install it
using ``pip``::

    pip install wsinfo

As an alternative you can get the source code from GitHub_ and install it
using the setup script::

    python setup.py install

Just check the installation::

    >>> import wsinfo
    >>> wsinfo.__version__
    '1.1.1'

And here we go!

.. note::
    The |wsinfo| library should be compatible with both Python 2 and 3.
